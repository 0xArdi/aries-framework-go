# Aries JS Worker

Aries Framework Go via Javascript.

## Build it

To build you will need:

* Go 1.13.x
* npm 6.13.x
* Node.JS 12.14.x
* bash

Run `npm install` from this directory. The output bundles will be placed in `dist/`.

View [`App.js`](./App.js) for examples.
